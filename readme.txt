== Description ==
Elementors Team Member Addon is an elementor add-on to showcase your Team Widget style/design. This is an simple and flexible way to add new elements/widgets to Elementor Page Builder.

Note: This plugin is a widget of Elementor Page Builder and will only work with Elementor Page Builder installed. This may have bugs and lack of many features. If you want to contribute to this project, you are more than welcome



== FEATURES ==

  8 Different Team Layout
  Unlimited Color Varition.
  Typography Options
  Designation Field
  Social Media Fields
  Support member Photo
  Support member Name
  Support member Title
  Support Any Social Link
  Drag & Drop Team Member 


== Frequently Asked Questions ==

1. Go to Plugins > Add New
2. Search Team Member Addon for Elementor
3. Click Install Now
4. Click Active Now
5. Go to Edit with Elementors Option & Select Elementor Teams Category
6. Click Team Member
